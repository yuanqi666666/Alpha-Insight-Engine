# Alpha Insight Engine 📈
这是一个基于 Python 和 WRDS 数据库的交互式金融深度分析引擎。

## 项目亮点
- **动态财务诊断**：根据实时数据自动评估公司的偿债能力与盈利健康度。
- **价值回测系统**：模拟特定历史节点的投资表现，直观展示持有至今的累计收益。
- **风险量化分析**：动态计算选定日期后的年化波动率与夏普比率（Sharpe Ratio）。
- **专业可视化**：包含 ROA/ROE/ROC、流动性比率及移动平均线等多维度图表。

## 如何使用
1. **环境准备**：确保本地已安装 `wrds`, `pandas`, `matplotlib`, `ipywidgets` 库。
2. **运行代码**：
   - 运行第一个单元格，在弹出框中输入您的 **WRDS 账号及密码** 进行登录。
   - 在交互界面输入 **美股代码**（如 AAPL, NVDA）并选择 **分析起始日期**。
   - 点击“生成深度价值报告”即可获取全方位分析。

## 技术栈
Python, SQL (WRDS Data), Pandas, Matplotlib, Ipywidgets
